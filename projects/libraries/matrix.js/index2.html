Matrix.js

Overview

A simple, fast and lightweight JavaScript library for doing mathematical operations on matrices, with no external dependencies.

You can create matrices and perform different types of calculations in code with very simple syntax. Matrix.js is fully free to use. And you can use it as you like.

Learn more about matrix.js by going down below


Get Started

To get started, you need to first download matrix.js. You can download matrix.js from here

After downloading matrix.js, use the code below inside your HTML code.

_(script src="matrix.js")__(/script)_

Matrix.js uses 'Matrix' as a namespace that contains all the functions. This is to avoid name collision with your code.
