Additional functions

This page contains additional functions, that you can use in matrix.js.


Matrix.identityOf(A) This function returns a new identity matrix of the same order as that of the given matrix. Parameters A: Object (matrix) Return Object (matrix)
Matrix.symmetricOf(A) This function returns a new symmetric matrix, that is related to the given matrix. The symmetric matrix is
B = A + A^{T}
Parameters A: Object (matrix) Return Object (matrix)
Matrix.skewOf(A) This function returns a new skew symmetric matrix, that is related to the given matrix. The skew symmetric matrix is
B = A - A^{T}
Parameters A: Object (matrix) Return Object (matrix)
Matrix.randomElement(A) Returns a random element of the given matrix. Parameters A: Object (matrix) Return Number
Matrix.generateRandomMatrix(order,minmax,toInt) This function generates a random matrix with the given parameters.
The parameters are:

Order: Which specifies the order of the random matrix. It should be an array of length 2, with the following format, [n,m]
minmax: Which specifies the minimum and maximum value, between which the random number should be generated. It's format is [min,max-1]. The default value is [0,(order[0]*order[1])+1]
toInt: Which specifies, whether to convert the values to an integer. The default value is true. If it is set to false, then all the values are floats.

//Generates a matrix of order 2 × 2. It's minimum value is 0 and maximum value is 9 -- var A = Matrix. generateRandomMatrix([2,2],[0,9],true);
Parameters order: array mimmax: array toInt: boolean Return Object (matrix)
Matrix.toTable(A) This function converts a matrix to a HTML table. It returns an object with a function 'addTo(element)' as a property.
The 'addTo(element)' takes in a HTML element as an argument. The HTML element is the place where it is created. For example:

var A = Matrix.create([1,2,3,4],2,2);---- //Creates a table element-- var table = Matrix.toTable(A);---- //Adds the table element to the body-- table.addTo(document.body);
