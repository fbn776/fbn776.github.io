This is a file that shows how to format the.

Each categorized feature in matrix.js is documented in a seperate file,
each having a header, side menu and a body.

To highlight code.
use,
<div class="codeElm"></div>

formatting inside codeElm:
use -- for <br>
use  (two spaces) for an indentation.
use _( for "<"
use )_ for ">"

Working:

'--' gets replaced with '<br>'
'  ' gets replaced with '<padd></padd>'
'_(' gets replaced with '&lt;' (<)
')_' gets replaced with '&gt;' (>)
