Determinants

This page contains functions to find the determinants of a matrix and other stuffs that are based on determinants


Matrix.det(A) Returns the determinant of a matrix Parameters A: Object (matrix) Return Number
Matrix.minors(A) Returns an array containing objects with the minors of a matrix at each position.
For example:

var A = Matrix.create([-- 1,2,-- 3,4 ],2,2);-- let arr = Matrix.minors(A);-- /*-- arr is:-- [-- {i:1,j:1,val:4},-- {i:1,j:2,val:3},-- {i:2,j:1,val:2},-- {i:2,j:2,val:1}-- ] */--
Parameters A: Object (matrix) Return Array
Matrix.minorsToMatrix(minors) This function converts the array of minors that is obtained from Matrix.minors() to a matrix and returns the new matrix.
For example:

var A = Matrix.create([-- 1,2,-- 3,4 ],2,2);-- let arr = Matrix.minors(A);---- let B = Matrix.minorsToMatrix(arr);-- /* B is:-- {-- n:2,-- m:2,-- matrix:[-- 4,3,-- 2,1-- ],-- }-- */
Parameters minors: Array Return Object (matrix)
Matrix.cofactors(A) This function returns an array containing cofactors of matrix. Works in the same way as Matrix.minors(). Parameters A: Object (matrix) Return Array
Matrix.cofactorsToMatrix(cofactors) This function converts the array of cofactors that is obtained from Matrix.cofactors() to a matrix and returns the new matrix.
For example:

var A = Matrix.create([-- 1,2,-- 3,4 ],2,2);-- let arr = Matrix.cofactors(A);---- let B = Matrix.cofactorsToMatrix(arr);-- /* B is:-- {-- n:2,-- m:2,-- matrix:[-- 4,-3,-- -2,1-- ],-- }-- */
Parameters cofactors: Array Return Object (matrix)
Matrix.adj(A) Returns the adjoint of a matrix, ie adjA Parameters A: Object (matrix) Return Object (matrix)
