Additional matrix operations

This page contains, some additional operations on matrices that, makes using matrix.js more simple


Matrix.neg(A) This function returns the negative of the given matrix. That is, this function returns the matrix that's multipled by -1. Parameters A: Object (matrix) Return Object (matrix)
Matrix.power(A,n) This function return the matrix that is raised by the power n.
For example:

let A = Matrix.create([1,2,3,4],2,2);-- //B = AA-- let B = Matrix.power(A,2);-- //B = AAA-- B = Matrix.power(A,3);--
Parameters A: Object (matrix) n: number (integer) Return Object (matrix)
Matrix.transpose(A) This function returns new matrix with interchanged row and column.
For example:

let A = Matrix.create([-- 1,2,-- 3,4-- ],2,2);-- let B = Matrix.transpose(A);-- /* B is:-- {-- matrix:[-- 1,3,-- 2,4-- ],-- n:2,-- m:2,-- }-- */
Parameters A: Object (matrix) B: Object (matrix) Return Object (matrix)
Matrix.inverse(A) This function returns the inverse of the matrix.
That is,
A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} \newline{} \newline{} A^{-1} = \ inverse \ of \ A \newline{}\newline{} A^{-1} \ is \ the \ matrix \ which \ satisfys \ the \ equation \newline{} A B = B A = I \newline{} where \ B = A^{-1}
Parameters A: Object (matrix) Return Object (matrix)
Matrix.identity(order) This function returns a new identity matrix with the given order. Parameters order: number (integer) Return Object (matrix)
