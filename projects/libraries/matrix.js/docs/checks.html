Types and Checks

This page contains all functions to check and validate different types of matrices.


Matrix.isMatrix(A) Check if the given argument is a matrix or not. Parameters A: Any type Return Boolean
Matrix.areMatrix(A,B...) Check if multiple entries are matrices or not Parameters A,B,C: Any types Return Boolean
Matrix.hasSameOrder(A,B) Checks if two matrices have same order. Parameters A: Object (matrix) B: Object (matrix) Return Boolean
Matrix.isEqual(A,B) Checks if two matrices are equal. That is, it checks if the two matrices has same order and each element in A at the position ij is equal to the element of B at ij Parameters A: Object (matrix) B: Object (matrix) Return Boolean
Matrix.canMult(A,B) Checks if two matrices can be multipled. That is, it checks if (A.m === B.n) Parameters A: Object (matrix) B: Object (matrix) Return Boolean
Matrix.isSquare(A) Checks if a matrix is a square matrix or not. Parameters A: Object (matrix) Return Boolean
Matrix.isDiagonal(A) Checks if a matrix is a diagonal matrix or a scalar matrix. Parameters A: Object (matrix) Return Boolean
Matrix.isIdentity(A) Checks if a matrix is an identity matrix Parameters A: Object (matrix) Return Boolean
Matrix.Row(A) Checks if a matrix is a row matrix Parameters A: Object (matrix) Return Boolean
Matrix.isCol(A) Checks if a matrix is a column matrix Parameters A: Object (matrix) Return Boolean
Matrix.isZero(A) Checks if a matrix is a zero matrix (null matrix) Parameters A: Object (matrix) Return Boolean
Matrix.isSymmetric(A) Checks if a matrix is a symmetric matrix. Parameters A: Object (matrix) Return Boolean
Matrix.isSkew(A) Checks if a matrix is a skew symmetric matrix Parameters A: Object (matrix) Return Boolean
Matrix.hasInverse(A) Checks if a matrix has an inverse. That is, it checks if det(A) != 0 Parameters A: Object (matrix) Return Boolean
