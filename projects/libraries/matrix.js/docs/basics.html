Basics of Matrices

Matrix.js uses a very simple and understandable matrix syntax. The main way to create a matrix is by its object notation, it's made simply by creating an object with 3 properties, namely

n: Number of rows of the matrix
m: Number of columns of the matrix
matrix: The matrix itself

var A = {-- n:2,-- m:2,-- matrix:[-- 1,2,-- 3,4,-- ]-- }


This code create a matrix of order 2 × 2. In mathematical terms,
A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}_{2Ã—2}
Matrix.create(elements,n,m) You can also create a new matrix using this function. It does the same thing as the above code. This returns a new matrix of order n × m Parameters elements: Array n: Number m: Number Return Object (Matrix)
Matrix.toMatrix(Array) You can alose represent matrix as 2 dimensional arrays. Where the outer array represents the rows and inner array represents the columns. Using this function, you can convert those types to the type that the library accepts.

/*Differnt way to represent a Matrix, but Matrix.js doesn't support this type*/-- var A = [-- [1,2],-- [3,4],-- ];-- /*Covert this to the Matrix.js equivalent*/-- A = Matrix.toMatrix(A);
Parameters Array: Array Return Object (Matrix)
Matrix.clone(A) Returns a new cloned matrix of the given matrix Parameters A: Object (matrix) Return Boolean
Matrix.extractMatrixByRow(A) This function takes each row of the the matrix and returns those rows in an array. For example:

var A = Matrix.create([-- 1,2,-- 3,4-- ],2,2);-- var arr = Matrix.extractMatrixByRow(A);-- /* arr is: [[1,2],[3,4]] */-- console.log(arr);

Parameters A: Object (matrix) Return Array
Matrix.extractMatrixByCol(A) This function takes each column of the the matrix and returns those columns in an array. For example:
var A = Matrix.create([-- 1,2,-- 3,4-- ],2,2);-- var arr = Matrix.extractMatrixByCol(A);-- /* arr is: [[1,3],[2,4]] */-- console.log(arr);
Parameters A: Object (matrix) Return Array
Matrix.expand(A) This function, expands a given matrix and returns an array of all the elements, with each element of array as an object with the following properties,
val: is the matrix element
i: the row number of the element,
j: the column number of the element.
For example:
var A = Matrix.create([-- 1,2,-- 3,4-- ],2,2);-- var arr = Matrix.expand(A);-- /* arr is:-- [-- {val:1,i:1,j:1},-- {val:2,i:1,j:2},-- {val:3,i:2,j:1},-- {val:4,i:2,j:2}-- ] */-- console.log(arr);
Parameters A: Object (matrix) Return Array
