Matrix operations

This page contains documentation on basic matrix operations such as addition, subraction, multiplication.


Matrix.add(A,B) Adds to matrices of same order and returns the resultant matrix. Parameters A: Object (matrix) B: Object (matrix) Return Object (matrix)
Matrix.sub(A,B) Subtract two matrices of same order and returns the resultant matrix. Parameters A: Object (matrix) B: Object (matrix) Return Object (matrix)
Matrix.multByScalar(A,c) Multiples a matrix with a constant (scalar) and returns the resultant matrix.
For example:

let A = Matrix.create([-- 1,2,-- 3,4-- ],2,2); let c = 2;---- let B = Matrix.multByScalar(A,c);-- /*-- B is:-- {-- n:2,-- m:2,-- matrix:[-- 2,4,-- 6,8,-- ]-- }-- */-- console.log(B);

That is, this code does the same as:
A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}_{2×2} \newline c = 2 \newline B = cA \newline ie, B = \begin{bmatrix} 2 & 4 \\ 6 & 8 \end{bmatrix}_{2×2}
Parameters A: Object (matrix) B: Object (matrix) Return Object (matrix)
Matrix.mult(A,B) This function multiples two matrices together and return the resultant matrix.
For example:

let A = Matrix.create([1,2,3,4],2,2);-- let B = Matrix.create([-1,-2,-3,-4],2,2);-- let C = Matrix.mult(A,B);-- /*-- C is:-- {n:2,m:2,-- matrix:[-7,-10,-15,-22]-- }-- */-- console.log(C);

That is, this code does the same as,
A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}\newline{} \newline{} B = \begin{bmatrix} -1 & -2 \\ -3 & -4 \end{bmatrix} \newline{} \newline{} C = AB \newline{} \newline{} ie, C = \begin{bmatrix} -7 & -10\\ -15 & -22 \end{bmatrix}
Parameters A: Object (matrix) B: Object (matrix) Return Object (matrix)
